# Gemini API Configuration Template

## API Key Setup Instructions

### Step 1: Get Your API Key
1. Visit https://ai.google.dev/
2. Sign in with your Google account
3. Click "Get API Key"
4. Create a new API key or use an existing one
5. Copy the API key (starts with "AIza")

### Step 2: Configure in the Application
1. Open the Gemini Image Editor in your browser
2. Paste your API key in the configuration section
3. Click "Save Key"
4. The key will be stored securely in your browser

### API Key Format
Your API key should look like this:
```
AIzaSyC-example-api-key-here-1234567890abcdef
```

### Security Notes
- Your API key is stored locally in your browser
- Never share your API key with others
- You can remove or change the key at any time
- The key is only sent to Google's Gemini API

### Troubleshooting
- If the key is rejected, double-check it starts with "AIza"
- Ensure there are no extra spaces when copying
- Verify the key is active in Google AI Studio
- Check your API usage limits in the Google Cloud Console

### Usage Limits
- Free tier: Limited requests per minute
- Check your quota at: https://console.cloud.google.com/
- Consider upgrading for higher usage needs
